pomodoro timer

A Pomodoro Timer is a productivity tool that helps manage time using the Pomodoro Technique. The technique is simple yet effective: it breaks work into intervals, traditionally 25 minutes in length, separated by short breaks (5 minutes). After completing four intervals, a longer break (15-30 minutes) is taken. Objective: Build a web-based Pomodoro Timer that helps users manage their work sessions, track productivity, and take timely breaks to avoid burnout. Key Features: Timer: A countdown for 25-minute work intervals (customizable). Short break (5 minutes) and long break (15-30 minutes) timers. User Interface: Clean, minimalistic, and intuitive design to start, pause, and reset the timer. Visual indication of the current session (work, short break, or long break).
